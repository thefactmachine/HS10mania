Economic Intelligence Report - Instructions

The following are some notes regarding the code and some instructions regarding how to run these reports.

What needs to be done to run the reports

The user will need to make sure there is a current set of data exported from TRED.

The R code currently does not automatically obtain the most recent data from TRED and this data will need to be obtained using a separate process.

Once you have the data...where to put it

The program expects to find an RDA file in the /inputs/ directory. The R file which reads this RDA file is called: "fn_read_data_and_convert_units.R" The user will need to open this code and change the line of code (i.e "load(...)") to the name of the RDA file obtained from TRED.

The program uses the following field names. If these names change, the program will fail.

  • "Harmonised_System_Code" (HS10 Codes)
  • "Unit_Qty" The type of units (i.e. Kilograms or Litres)
  • "Country" Export country (i.e. Australia)
  • "Total_Exports_NZD_fob" Value in $NZ of the exports
  • "Total_Exports_Qty" Number of units exported
